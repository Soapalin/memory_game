# Memory Match

A small, static memory card matching game built with plain HTML, CSS, and JavaScript. It runs in any modern browser with no build step or dependencies.

## Features
- Multiple themes and difficulty sizes
- Move counter, timer, and best score saved per difficulty
- Keyboard navigation with arrow keys and Enter/Space
- Match and mismatch feedback with a brief lockout

## Run locally
1. Open `index.html` in your browser.
2. Click New Game or change theme/difficulty to restart.

## How to play
- Flip two cards to find a pair.
- Matched cards stay revealed.
- Mismatched cards flip back after a short delay.

## Notes
- Best scores are stored in localStorage per difficulty.
- No network access or external assets are required.

## Project structure
- `index.html`: all HTML, CSS, and JavaScript for the game.
